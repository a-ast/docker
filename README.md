# Docker Containers for Web Development


## Contents

 - **/docker-compose.yml**: used to orchestrate the Dockerfiles
 - **/etc:** Dockerfiles and their configurations.
 - **/var/www**: Your web files live here.

## Requirements

Install Docker and Docker Compose.
 
## Usage

```sh
$ docker-compose up -d
```
It's that simple!
